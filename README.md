# مغسلة السجاد - تطبيق أندرويد (Flutter)

## نظام إدارة طلبات مغسلة السجاد مع تتبع المناديب

---

## المتطلبات

- **Flutter SDK**: 3.2.0 أو أحدث
- **Android Studio**: أحدث إصدار مع Android SDK 34
- **Min SDK**: 24 (Android 7.0)
- **Target SDK**: 34 (Android 14)

## التثبيت

### 1. تثبيت Flutter
```bash
# تحميل Flutter من الموقع الرسمي
https://docs.flutter.dev/get-started/install

# التحقق من التثبيت
flutter doctor
```

### 2. تثبيت المكتبات
```bash
cd flutter_laundry_app
flutter pub get
```

### 3. إعداد Firebase
1. أنشئ مشروع Firebase جديد
2. أضف تطبيق أندرويد:
   - Package Name: `com.laundry.app`
   - SHA-1: من `flutter app signing` في Android Studio
3. حمّل ملف `google-services.json` وضعه في:
   ```
   android/app/google-services.json
   ```
4. فعّل Cloud Messaging في Firebase Console

### 4. تشغيل التطبيق
```bash
# تشغيل في وضع Debug
flutter run

# بناء ملف APK
flutter build apk --release

# بناء App Bundle للنشر
flutter build appbundle --release
```

## الميزات

### إدارة الطلبات
- إضافة طلبات جديدة مع صورة المنزل
- تعديل وتحديث حالة الطلب
- فلترة وبحث متقدم
- تصدير Excel
- إرسال واتساب تلقائي

### إدارة المندوبين
- إضافة وتعديل مناديب
- تتبع GPS مباشر على الخريطة
- إرسال إشعارات للمندوبين
- عرض طلبات كل مندوب

### لوحة التحكم
- إحصائيات فورية (انتظار/تسليم/أرباح/أمتار)
- طلبات اليوم الشاملة
- رسوم بيانية متحركة
- تنبيهات فورية

### المصروفات والرواتب
- إدارة مصروفات يومية
- مصروفات المدير
- رواتب العمال
- تقارير مالية شهرية

### التقارير
- رسوم بيانية متنوعة (خطوط/أعمدة/دائرية)
- تقارير شهرية وسنوية
- تصدير PDF
- تقارير المندوبين

### الإعدادات
- تخصيص ألوان التطبيق
- إعدادات الفاتورة
- قالب رسالة واتساب
- النسخ الاحتياطي
- رفع شعار وأيقونة المغسلة

### الإشعارات
- إشعارات فورية (Firebase)
- إشعارات محلية
- صوت تنبيه مخصص
- إشعارات الطلبات الجديدة
